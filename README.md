# Naval-Battle
###### Jogo de Batalha naval contra Inteligencia artificial

### O jogo
Nessa aplicação de batalha naval, inicialmente, o usuário deve ter diferentes tamanhos de "barcos" para posicionar na tela de maneira horizontal e vertical.
Após montar o seu board, começam as rodadas de tiros, onde o usuário escolhe um lugar para atirar no board da "inteligencia artificial", caso acertar, ele pode repetir a jogada.
A mesma coisa acontece para a inteligencia artificial.

### A inteligencia artificial
Como o conhecimento nesse área é um pouco complexo. Criei uma inteligencia artificial "burra" que monta seu board aleatoriamente e atira aleatoriamente no board humano.
